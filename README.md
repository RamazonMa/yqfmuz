Har bir sahifada:

- Tashkilot logotipi va O'zbekiston Davlat gerbi
- F.I.Sh., lavozim va tashkilot nomi
- Manzil (Yandex Xaritalar havolasi), ish va mobil telefon raqamlari, elektron pochta
- O'zbekcha / Inglizcha til almashtirish (yuqori o'ng burchakdagi tugma)
- Mobil va desktop qurilmalar uchun moslashuvchan (responsive) dizayn

## Foydalanish

Har qanday HTML faylni brauzerda ochish kifoya — qo'shimcha o'rnatish yoki build jarayoni talab qilinmaydi.

GitHub Pages orqali joylashtirish uchun repo sozlamalarida Pages manbasini yoqib, `xodimlar/<fayl_nomi>.html` manziliga murojaat qilish mumkin.

## Texnologiyalar

- HTML5, CSS3 (custom properties, gradient, flexbox)
- [Font Awesome 4.7.0](https://fontawesome.com/v4/) (CDN)
- Vanilla JavaScript (til almashtirish funksiyasi)
